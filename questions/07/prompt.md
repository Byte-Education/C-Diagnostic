# Question 7

Write a function that checks if two strings are the same case-insensitive

    Example output: 

    printf("%d\n", equals("hello", "hello")); // 1
    printf("%d\n", equals("hello", "Hello")); // 1
    printf("%d\n", equals("hello", "hellp")); // 0
    printf("%d\n", equals("hello", "Hellp")); // 0
